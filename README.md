# Web Application "Filmoteka"

Team project on JavaScript, HTML5, SCSS

## Requirements
- JavaScript
- HTML5/SCSS
- Webpack
- NPM
## Installation
```bash
git clone https://github.com/OlegGap/project_js
```
```bash
cd ./project_js
```
```bash
npm install
```
## Development and Deployment
```bash
npm start #Start development
```
```bash
npm run build #Build to production
```
